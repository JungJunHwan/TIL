# 자바가 확장한 객체 지향

## JUnit의 @BeforeClass

- JUnit에서 테스트 클래스가 실행되기 전 딱 한 번만 실행되는 메소드에 붙는 어노테이션
- 테스트 메소드들 실행 전에 클래스 단위로 한 번, 공용 초기화 작업을 수행하기 위한 기능
- JUnit5 부터는 @BeforeAll로 바뀌었다
- 해당 어노테이션이 붙는 메소드는 static 메소드여야한다

## 인스턴스 블록

- new로 객체가 생성될 때마다 실행되는 초기화 블록
- 클래스 안에 아래와 같이 써놓으면 인스턴스 초기화 블록이 된다

```java
{

}
```

- 생성자가 실행되기 전에 실행된다
- 객체 생성 순서
    1. 필드 초기화
    2. 인스턴스 초기화 블록 실행
    3. 생성자 실행
